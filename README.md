# vehicle-research
This is a repository I created for buying a new vehicle. 
A program written in python will web scrape the website kijiji.ca for vehicles that match my personal buying criteria. 
This will drastically cut down on loading times required for setting filiters on Kijiji. 
A simple run of this program can give me new information about the auto market quickly. 
I can simply read key elements of the ad in text within the text files for each assoicated brand and model.
